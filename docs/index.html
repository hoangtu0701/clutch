What it does

Clutch listens for “Hey Jarvis”, transcribes your voice in real time, reads CS2 Game State Integration (GSI) + targeted screenshots, reasons with an LLM, and streams back spoken advice via low-latency TTS (Google Chirp 3 HD / Coqui XTTS / Piper).

  • Sub-second “first audio” with streaming pipeline
  • Wake-word + VAD + STT using RealtimeSTT/Whisper + Silero
  • Local Flask GSI server + multi-region screen capture (OpenCV)
  • OpenAI Responses API with OpenRouter fallback
  • PyQt5 desktop UI, thread-safe queues/locks, optional CUDA
