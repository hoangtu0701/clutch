How it works

  1. 1

    Wake

    Say “Hey Jarvis”. OpenWakeWord + VAD starts the capture.

  2. 2

    Hear

    Realtime STT streams partials so you see what’s heard instantly.

  3. 3

    See

    GSI + targeted screenshots (radar, alive counts) anchor the advice.

  4. 4

    Think

    LLM (OpenAI Responses API with OpenRouter fallback) reasons in context.

  5. 5

    Speak

    Streaming TTS talks as tokens arrive. Interrupt anytime to re-ask.
