How it works

  1. 1

    Wake

    Say “Hey Jarvis”. OpenWakeWord + Silero VAD starts listening.

  2. 2

    Hear

    Realtime STT streams partials for responsiveness.

  3. 3

    See

    Clutch reads CS2 GSI + captures targeted UI regions (radar, alive counters).

  4. 4

    Think

    LLM (OpenAI via Responses API, with OpenRouter fallback) reasons in context.

  5. 5

    Speak

    Streaming TTS speaks tokens ASAP. Interrupt any time with “Hey Jarvis”.
